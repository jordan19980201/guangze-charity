# 新竹市廣澤慈善協會 官方網站 — 設計文件

- **日期**：2026-06-02
- **狀態**：已由使用者口頭核准，授權自主建置
- **參考對象**：charity: water（charitywater.org）的版面結構與敘事風格

---

## 1. 專案目標

為「新竹市廣澤慈善協會」建立官方形象網站。三大重點：

1. **建立組織形象與信任** — 介紹協會宗旨、沿革、理事長、透明度
2. **活動公告** — 公布「百斤米一世情」等活動資訊
3. **成果展示與故事分享** — 用照片與真實故事呈現影響力

**非目標（第一版不做）**：線上募款金流、線上活動報名、會員系統、多語系（僅繁體中文）。

---

## 2. 組織背景（取自新聞報導與臉書）

- **全名**：新竹市廣澤慈善協會（統一編號 50598756）
- **理事長／創辦人**：曹禎祥
- **宗旨**：「取之社會、用之社會」，扶弱濟貧
- **招牌活動**：「百斤米、一世情」慈善捐助，連續舉辦十多年，每年嘉惠新竹、竹苗地區約 1,000 戶弱勢家庭
- **服務對象**：中低收入戶、持身心障礙手冊者
- **物資**：白米、麵條、罐頭、乾糧
- **附加服務**：愛心餐點、義剪、視力檢查、血糖篩檢、X 光肺部檢查
- **合作夥伴**：理髮店、家具公司、餐廳、眼鏡公司、獅子會、衛生單位、警方（防詐宣導）
- **據點**：新竹市指澤宮
- **其他善行**：捐資興學、防詐騙宣導

> 註：部分數據（成立年份、累計捐贈量）尚待協會確認，網站中先以可於後台編輯的欄位呈現。

## 4. 網站架構（頁面）

1. **首頁** — 主視覺輪播、使命、影響力數據、招牌善行、最新消息、行動呼籲
2. **關於我們** — 宗旨、沿革、理事長的話、組織理念
3. **我們的善行** — 百斤米一世情、義剪、健康篩檢、捐資興學、防詐宣導
4. **最新消息／活動公告** — ⭐ 後台可新增（列表＋內頁）
5. **成果與故事** — ⭐ 後台可新增（照片＋受助故事）
6. **響應愛心** — 如何捐款／捐物資／當志工（附聯絡方式）
7. **聯絡我們** — 地址（指澤宮）、電話、LINE、臉書、地圖

## 5. 首頁區塊（由上到下）

1. **主視覺 Hero** — **全螢幕照片輪播**，數張大圖自動淡入淡出輪流播放；疊上 Logo、標語「取之社會，用之社會」、行動按鈕
2. **一句話使命** ＋ 簡短介紹
3. **影響力數據帶** — 例：連續 N 年｜每年 1,000 戶｜累計白米 ◯◯ 斤（數字後台可編輯）
4. **招牌善行：百斤米一世情** — 圖文特寫
5. **我們做的事** — 服務卡片（物資／義剪／健康篩檢／興學／防詐）
6. **最新消息** — 自動帶出後台最新 3 篇
7. **響應愛心** 行動呼籲帶
8. **頁尾** — 聯絡資訊、臉書、地圖

## 7. 技術架構

> ⚠️ **環境註記**：本機未安裝 Node.js。若 Node 安裝成功，採「方案 I：Astro」；若無法安裝，採「方案 II：免建置純靜態」。兩者皆達成上述所有功能與後台需求，差別僅在建置方式。最終採用方案將於施工計畫中註明。

### 方案 I（首選）：Astro + Sveltia CMS
- **網站產生器**：Astro（內容型網站、圖片優化佳）
- **樣式**：Tailwind CSS
- **後台**：Sveltia CMS（`/admin`，GitHub 帳號登入）
- **內容**：Markdown/JSON content collections
- **部署**：GitHub → Cloudflare Pages（免費，自動建置）

### 方案 II（備案）：免建置純靜態網站
- **前端**：手寫 HTML + CSS + 原生 JS（無建置步驟）
- **後台**：Sveltia CMS（`/admin`，GitHub 帳號登入），內容存為 JSON/Markdown 並 commit 進 repo
- **動態內容**：消息、故事、輪播圖以 JS 於前端 fetch JSON 後渲染，新增內容免重新建置即可顯示
- **部署**：GitHub → Cloudflare Pages（免費，無需建置指令）
- **取捨**：消息／故事頁為前端渲染，SEO 略弱於方案 I；主要靜態頁（首頁、關於、善行）仍為靜態 HTML，SEO 正常

### 兩方案共通
- 程式碼與內容置於 GitHub 儲存庫
- 後台存檔 → commit → 自動部署上線
- 後台登入需使用者註冊免費 GitHub 帳號（建置者協助設定 OAuth）
